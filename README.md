Order Track

> **Abstract** : Problem Statement
Current manual order management in local shops leads to significant human error, time-consuming search processes for product data, and excessive paper usage. There is a lack of real-time stock monitoring, making it difficult to generate accurate daily or weekly sales reports. This project aims to digitize these records to enhance efficiency and reliability.
The proposed solution is a web-based  Shop Management System that digitizes the entire sales lifecycle. It replaces manual registers with a centralized database to automate order tracking, inventory updates, and report generation.
